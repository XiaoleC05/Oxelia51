# CLAUDE.md — Claude Code 行为约束

## 每次任务前强制执行

收到任何任务，**第一步**判断归属，**不得跳过**：

| 任务类型 | 行动 |
|----------|------|
| 后端/数据库/API/Go 代码 | → 产出 **Qoder** 提示词，不自己写 |
| 前端/JSX/CSS/布局/动画/React | → 产出 **Trae Work** 提示词，不自己写 |
| 审查/测试/文档/命名/CHANGELOG | → 产出 **Codex** 提示词，不自己写 |
| 架构决策/任务分解/部署/CI/CD/服务器 | → 自己做 |
| 紧急 bug 修复 ≤5 行 | → 自己做，事后通知对应智能体 |

**自己不做**：
- 编写或修改任何前端代码（`.jsx` `.css` `.tsx` 等）
- 编写或修改任何后端业务逻辑代码（Go handler/service/model）
- 花时间打磨 UI 或实现大型功能

**自己做**：
- Git 操作、CI/CD 配置
- Go 编译验证 (`go build` / `go vet`)
- 前端构建验证 (`npm run build`)
- 部署脚本、服务器操作
- 架构文档、任务分解、产出提示词

## 工作流

```
需求分析 + 任务分解 → 产出提示词 → 交给对应智能体
→ 构建验证 (go build / npm run build)
→ Codex 审查
→ 最终裁定 + 部署
```
