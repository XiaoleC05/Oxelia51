# Oxelia51 — 开发者工具平台

> 一个入口，畅享全部工具。一个账号，无缝切换。

## 这是什么

Oxelia51 是一个聚合型的在线工具平台。和零散的工具站不同——你只需注册一次，就能使用平台上所有工具。所有操作都在平台内完成，前端统一渲染，后端自动将请求转发到对应工具的服务，你感觉不到任何跳来跳去的割裂感，就像在用一个大应用。

## 怎么跑起来的

```
你的浏览器 → oxelia51.com
                ├── React 前端（所有页面统一渲染）
                ├── Go 后端（登录注册、工具管理、请求转发）
                ├── PostgreSQL（用户信息、工具配置）
                └── Redis（登录状态缓存、任务队列）
                       │
                       ▼ 内网转发到各工具
                ┌──────┼──────┬──────┐
              工具A   工具B   工具C   工具D
             (内网)   (内网)   (内网)   (内网)
```

每个工具只提供后端接口，没有独立的前端页面。工具端口不暴露到公网。

## 技术选型

| 层级 | 用什么 |
|------|--------|
| 后端 | Go + Gin |
| 前端 | React（Vite） |
| 数据库 | PostgreSQL 17 |
| 缓存 | Redis 7 |
| 部署 | Docker Compose + Nginx |
| 接口风格 | REST |

## 怎么用

- **线上**：打开 [oxelia51.com](https://oxelia51.com)，注册就能用
- **本地**：去每个工具的 GitHub Releases 下载 exe，双击运行就行
