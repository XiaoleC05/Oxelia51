# ADR-004: 工具项目清单对齐本地 `code` 目录

**日期**：2026-07-05  
**状态**：已采纳

---

## 背景

1. 问卷阶段使用的工具名 `dorm-power-guard-lite-repo` 已更名为 **DormGuard**（仓库 `github.com/XiaoleC05/DormGuard`）。
2. 早期文档中的概念工具（文件转换器、RSS 阅读器、短链接、临时文件分享）已落实为 `D:\07_Projects\code` 下的具体项目，或不再纳入计划。
3. 需将 Oxelia51 文档中的工具清单与本地实际目录保持一致。

## 决策

**`D:\07_Projects\code` 目录下的工具项目即为 Oxelia51 当前已定稿的在线工具（及关联资产）清单。**

| 项目 | 本地路径 | GitHub 仓库 | 平台角色 |
| --- | --- | --- | --- |
| Oxelia51 | `D:\07_Projects\code\Oxelia51` | `XiaoleC05/Oxelia51` | 平台主项目 |
| DormGuard | `D:\07_Projects\code\DormGuard` | `XiaoleC05/DormGuard` | 在线工具（slug: `dormguard`，路径 `/tools/dormguard`） |
| SuperRead | `D:\07_Projects\code\SuperRead` | `XiaoleC05/SuperRead` | 在线工具（RSS + AI 简报） |
| AIHelper | `D:\07_Projects\code\AIHelper` | `XiaoleC05/AIHelper` | 在线工具（提示词助手） |
| AgentCanvas | `D:\07_Projects\code\AgentCanvas` | `XiaoleC05/AgentCanvas` | 在线工具（Agent 可视化） |
| XiaoleC05.github.io | `D:\07_Projects\code\XiaoleC05.github.io` | `XiaoleC05/XiaoleC05.github.io` | 独立个人站，平台底部链接引流 |

### 更名说明

| 旧名称 | 新名称 | 说明 |
| --- | --- | --- |
| `dorm-power-guard-lite-repo` | **DormGuard** | 宿舍电费监控工具，仓库与本地目录均已更名 |

### 不再纳入计划的早期概念

以下条目仅存在于问卷阶段，已被上表中的实际项目替代或移除：

- `file-format-converter`（文件转换器概念）
- `rss-aggregator-reader`（已由 **SuperRead** 承接）
- `short-link-platform`（短链接）
- `temporary-file-sharing`（临时文件分享）

## 理由

- 文档与代码仓库保持同步，避免新会话或协作者引用过时项目名。
- 明确在线工具清单与 slug，便于网关与扫描实施。
- 本地 `code` 目录即项目扫描的目标范围，清单固定后便于阶段2（手动扫描）实施。

### 修订（2026-07-05）

- DormGuard 纳入在线工具，与其他工具同等接入平台（ADR-006 规定现网首页过渡策略）。
- 作品集改由 `/portfolio` 展示 `code` 目录项目，不再用 DormGuard 充当作品集案例。

## 影响

- `docs/` 下需求与 API 文档已同步。
- DormGuard 经 `/tools/dormguard` 接入；现网首页过渡见 ADR-006。
