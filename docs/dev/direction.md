## 二、需要添加的

### 2.1 首页 — AI 大模型能力榜单（核心新功能）

**定位**：AI 时代开发者每天都要选模型。聚合权威数据，用多维度对比帮助访客做选择。

**数据来源**（11 个权威平台）：

| 数据源 | 测什么 |
|--------|--------|
| LMSYS Chatbot Arena | 人类盲评综合排名（Elo 分） |
| SWE-bench Verified | 真实 GitHub issue 修复能力 |
| LiveCodeBench | 代码生成（时效性强，防数据污染） |
| Aider Polyglot | 多语言代码编辑能力 |
| Artificial Analysis | 性价比、速度、延迟 |
| OpenRouter Rankings | 多维度社区评分 |
| WebArena | Web 任务（前端操作能力） |
| C-Eval / CMMLU | 中文知识与推理 |
| SuperCLUE | 中文综合能力 |
| BigCodeBench | 代码理解与生成 |
| RULER / LongBench | 长上下文能力 |

**榜单维度**（8 个）：

| 维度 | 回答的问题 |
|------|-----------|
| 综合编程 | 哪个写代码最强？ |
| 前端能力 | 哪个写 React/CSS 最好？ |
| 后端能力 | 哪个写 Go/Python/DB 最好？ |
| 代码审查 | 哪个找 bug 最准？ |
| 性价比 | 哪个能力/价格最优？ |
| 速度 | 哪个输出最快（tokens/s）？ |
| 中文能力 | 哪个中文理解最好？ |
| 长上下文 | 哪个能处理最长代码库？ |

**展示形态**：每维度一张卡片，Top 5 排名 + 分数 + 进度条。

**更新策略**：自动化脚本每周拉取，Redis 缓存，记录历史趋势。

### 2.2 首页 — 有趣网站推荐

**定位**：每周一期，推荐 5-10 个真正值得看的网站。访客会回来看看"这周推荐了什么"。

**分类**：开发者工具 / 设计灵感 / 独立博客 / AI 工具 / 有趣互动 / 数据可视化 / 极简之美

**运营**：你推荐 + 用户投稿（审核后发布）+ 历史归档浏览

### 2.3 用户个人主页系统

**核心产品**：`/@username` — 每个注册用户拥有自己的个人主页。

**功能**：
- 选择模板（温馨/星空/更多）
- 填写信息（姓名、技能、项目、社交链接）
- 可选模块（博客、时间线、友情链接）
- 自定义域名（后期）

### 2.4 用户自助后台

每个注册用户可访问：
- 编辑个人信息
- 管理项目/作品
- 选择/切换模板
- 查看访问统计

### 2.5 管理后台（仅 oxelia51）

保留并扩展：

| 功能 | 状态 |
|------|:--:|
| 用户管理（查看/编辑/删除） | 已有 |
| 内容管理（博客/头图/作品集） | 已有 |
| 服务器监控（阿里云+腾讯云） | 已有 |
| 榜单数据管理（手动更新/审核） | **新增** |
| 有趣网站投稿审核 | **新增** |
| SecretStore（管理员加密保险箱） | 已有，保留 |
| DormGuard（管理员电费监控+QQ机器人） | 已有，保留 |

### 2.6 前端设计系统

保留已有的：
- 双主题设计（Cozy/Cosmos，已设计完毕）
- Canvas 粒子背景
- 8px 网格 + 四原则（对齐/亲密/对比/重复）
