# Oxelia51 平台使用指南

Oxelia51（奥泽莉亚51）是一站式开发者工具平台。注册账号后即可在线使用全部工具。

## 注册与登录

1. 打开 [oxelia51.com](https://oxelia51.com)，点击右上角「注册」
2. 填写账号 ID（字母数字下划线）、用户名、邮箱、密码
3. 注册后查收验证邮件，点击链接激活账号
4. 验证后即可登录

## 浏览工具

1. 点击导航栏「工具」进入工具目录
2. 卡片显示工具名称、描述和状态徽章（已开放 / 暂未开放 / 已下线）
3. 点击卡片或「进入工具」按钮，在平台内直接使用

## 博客与资源

- **博客**：[oxelia51.com/blog](https://oxelia51.com/blog) — 技术文章
- **关于开发者**：[oxelia51.com/about](https://oxelia51.com/about) — 开发者简介与联系方式
- **友情链接**：[oxelia51.com/friends](https://oxelia51.com/friends) — 推荐站点

## 个人设置

1. 登录后点击右上角用户名 →「资料」可修改显示名
2. 「密钥」页面可管理 API Key（用于工具外部调用）

## 常见问题

**Q: 忘记密码？**
登录页点击「忘记密码」，输入注册邮箱，按邮件提示重置。

**Q: 未收到验证邮件？**
检查垃圾邮件箱。登录页可点击「重新发送验证邮件」。

**Q: 某些工具显示「暂未开放」？**
该工具当前仅对管理员开放，关注博客获取上线通知。
