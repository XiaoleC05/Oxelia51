# ADR-001: 开放公开注册

**日期**：2026-06-08  
**状态**：已采纳

---

## 背景

原始决策（2026-06-03）记录「MVP 阶段不开放公开注册」。当前实际已实现 `POST /api/auth/register` 接口和 React 注册页面（阶段3/4产出），与决策不一致。

## 决策

推翻原决策，改为 **允许公开注册**。

## 理由

- 已投入开发成本：注册接口和前端页面均已完成并测试通过
- MVP 目标包含「完整 MVP（访问者入口 + 登录 + 管理员界面）」——公开注册是实现该目标的合理组成部分
- 无技术或安全层面不可接受的风险（可在后续版迭代中加入邀请码/审核机制）

## 影响

- `implementation-specification.md` 已同步更新
- 现有代码无需回退
- 后续可考虑增加注册限制（如邀请码），作为功能迭代而非移除
