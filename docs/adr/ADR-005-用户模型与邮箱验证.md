# ADR-005: 用户模型、邮箱验证与工具可见性

**日期**：2026-07-05  
**状态**：已采纳（v1.1 修订）  
**修订**：`visitor_visible` → `user_accessible`；限流独立计数；邮箱 UNIQUE

---

## 背景

平台面向：未登录访客、已注册普通用户、唯一管理者 `oxelia51`。未登录不可使用在线工具。普通用户仅可用管理者开启的在线工具；管理者可用全部已启用且具备在线能力的工具。

## 决策

### 用户角色

| 角色 | 说明 |
|------|------|
| 访客 | 可浏览公开页、作品集、工具目录；不可用在线工具 |
| 未验证用户 | 不可登录、不可用在线工具 |
| 普通用户 `role=user` | 邮箱已验证；仅 `user_accessible=true` 且 `status=enabled` 的工具 |
| 管理者 `oxelia51` `role=admin` | 无视 `user_accessible`；受 `status` 与 `online_capable` 约束 |

### 注册与认证

- 开放注册：用户名、密码、确认密码、邮箱。
- **`email` UNIQUE**；一邮箱一账号（审查 M6）。
- 注册 → 验证邮件 → 验证后方可登录。
- 登录：用户名 + 密码；access JWT **7 天**；refresh **30 天**；`POST /api/auth/logout` 黑名单。
- 忘记密码：独立限流，同邮箱 ≤1/天。

### 限流（独立计数）

| 操作 | 限制 |
|------|------|
| 注册 | 同 IP ≤3/小时 |
| 重发验证 | 同邮箱 ≤1/天 |
| 忘记密码 | 同邮箱 ≤1/天 |
| 登录失败 | 同 IP ≤10/15min |

### 工具可见性字段

- 使用 **`user_accessible`**（废弃 `visitor_visible`）。
- 整工具开关；仅约束普通用户。
- UI 徽章规则见 platform-api.md §4.1。

### 管理者 `oxelia51`

- 迁移种子；`ADMIN_INITIAL_PASSWORD` 或首次自动生成 + 桌面 `oxelia51-admin-password.txt`。

### 邮件

- 生产：阿里云邮件推送；Token 24h Redis 单次有效。

## 影响

- 所有契约文档 v1.1 同步字段重命名。
- `users.email` UNIQUE 约束写入 migration。
