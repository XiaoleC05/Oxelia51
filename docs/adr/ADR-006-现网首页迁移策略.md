# ADR-006: 现网首页迁移策略（DormGuard → /tools/dormguard）

**日期**：2026-07-05  
**状态**：已采纳（v1.1 修订：Nginx 共存）  
**修订**：回应 Codex 契约审查 P0（D1）

---

## 背景

- 当前 `https://oxelia51.com/` 直接指向 **DormGuard** 独立部署（试验阶段，日常仍依赖）。
- 现网 Nginx 将 **全部** `location /api/` 转发至 DormGuard `:8000`。
- 平台 Go API 同样使用 `/api/auth/*`、`/api/tools/*` 等前缀。
- 目标架构：平台落地页占 `/`；DormGuard 位于 `/tools/dormguard`，经网关访问。

## 决策

### 阶段划分

| 阶段 | 公网 `oxelia51.com` | 平台 | DormGuard API |
|------|---------------------|------|---------------|
| **0（当前）** | `/` → DormGuard 前端；`/api/` → `:8000` | 仅本地 `:8080` | 公网可达（经 Nginx） |
| **1** | **不变** | 本地 + SSH 隧道访问 `:8080` / `:5173` | 公网仍 `/api/` → `:8000`；网关经 **内网** `127.0.0.1:8000` 转发 |
| **2（切换日·须开发者同意）** | `/` → 平台前端；`/api/` → 平台 Go `:8080` | 公网上线 | **仅** `127.0.0.1:8000` 监听；无公网直连 |

### 阶段 0～1：Nginx 共存策略（阻断项解决方案）

**公网 Nginx 不接入平台 Go 服务。** 现网配置保持：

```nginx
# 现网保持不变（阶段 0～1）
location /api/ {
    proxy_pass http://127.0.0.1:8000/api/;   # DormGuard
}
location / {
    proxy_pass http://127.0.0.1:8000/;       # DormGuard 前端或静态
}
```

**平台独立端口（仅本机 / SSH）：**

| 服务 | 端口 | 访问方式 |
|------|------|----------|
| 平台 Go | `:8080` | `localhost:8080`、`ssh -L 8080:127.0.0.1:8080` |
| 平台 Vite | `:5173` | 本地开发；`vite.config` 代理 `/api` → `:8080` |

阶段 1 网关转发 DormGuard 时：`internal_api_base = http://127.0.0.1:8000`（走 loopback，**不经过**公网 Nginx）。

### 阶段 2（切换日）目标 Nginx 示例

```nginx
# 切换日后（示意，实施日以运维文档为准）
location /api/ {
    proxy_pass http://127.0.0.1:8080/api/;   # 平台 Go（含网关）
}
location / {
    proxy_pass http://127.0.0.1:5173/;       # 或平台静态构建产物
}
# DormGuard :8000 仅 bind 127.0.0.1，无独立 location
```

### 原则

1. **切换日前不得破坏现网 DormGuard** 的日常使用。
2. DormGuard slug 固定 **`dormguard`**。
3. 阶段 1 管理者通过 **SSH 隧道** 内测平台，不与公网 DormGuard 抢 Nginx。
4. DormGuard 内网化改造见 [ADR-007](ADR-007-DormGuard内网化与数据库边界.md)。

### 阶段 1 DormGuard 工具登记状态

见 ADR-007 §6：`user_accessible=false`，`status=enabled`，`online_capable=true`。

## 理由

- 审查项 D1：两套 `/api/` 无法在同一 Nginx `location` 共存；端口隔离是阶段 1 唯一安全方案。
- Q19:A（不公网暴露未完成平台）与现网保护一致。

## 影响

- Trae/Cursor 阶段 1 联调须用 SSH 隧道或纯本地，不依赖 `oxelia51.com` 公网访问平台 API。
- 切换日需一次性 Nginx 切换 + DormGuard bind 地址改为 loopback。
