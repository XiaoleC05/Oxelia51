# Oxelia51 需求确认摘要

**版本**：v2.0  
**日期**：2026-07-05  
**修订**：Codex 契约审查修订版

## 1. 平台定位

- 统一在线工具平台；API 网关转发各工具后端。
- 作品集 `/portfolio`：`code` 目录全部项目，名称+描述（`portfolio_items` 表，可后台覆盖）。
- 桌面安装包：所有人可点 GitHub Releases。

## 2. 路径

| 路径 | 说明 |
|------|------|
| `/` | 现网 DormGuard；切换后平台落地页 |
| `/portfolio` | 作品集 |
| `/tools` | 在线工具目录 |
| `/tools/dormguard` | DormGuard |
| `/tools/:slug` | 其他工具 |
| `/login` `/register` `/verify-email` `/reset-password` | 认证 |
| `/admin/*` | 仅 `oxelia51` |

## 3. 用户与权限

| 角色 | 在线工具 |
|------|----------|
| 未登录 / 未验证 | 否 |
| 普通用户 | 仅 `user_accessible=true` 且 `status=enabled` |
| `oxelia51` | 无视 `user_accessible`；须 `status=enabled` |

- 工具目录：展示全部在线工具 + 徽章（已开放 / 暂未开放 / 已下线）。
- 字段 **`user_accessible`**（非 visitor_visible）：约束**普通用户**，不是访客。

## 4. 注册 / 认证 / 限流

- 开放注册；用户名+密码+确认密码+邮箱；**邮箱唯一**。
- 验证邮件 24h；验证后可登录。
- 忘记密码 + 重置（本期）。
- JWT：access 7d，refresh 30d，logout 黑名单。
- 限流（**独立计数**）：
  - 注册：同 IP ≤3/小时
  - 重发验证：同邮箱 ≤1/天
  - **忘记密码：同邮箱 ≤1/天**
  - 登录失败：同 IP ≤10/15min

### 邮件与管理者

- 生产：**阿里云邮件推送**
- 管理者 **`oxelia51`**；种子或 `ADMIN_INITIAL_PASSWORD` 或首次生成写桌面

## 5. 工具与 DormGuard

- 6 个在线工具 + slug 见 tool-registration.md。
- DormGuard：独立 MySQL；平台 PostgreSQL；网关 loopback 转发（ADR-007）。
- 现网 `oxelia51.com/` + `/api/` → DormGuard **不变**（阶段 0～1）。
- 平台 `:8080` 仅本地/SSH（ADR-006）。
- 阶段 1 DormGuard：`user_accessible=false`，`status=enabled`。
